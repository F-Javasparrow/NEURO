# 神经损伤 - Neurotrauma

~~指能对玩家造成神经损伤的模组~~

灵感来源于游戏 潜渊症(Barotrauma) 的同名模组

## 核心功能

* [X] 症状/药物框架 (添加新症状 见 [Add Symptoms](https://github.com/F-Javasparrow/Neurotrauma/docs/addSymptom.md))
* [X] 症状/药物相互作用
* [ ] 药代动力学
* [X] 医疗界面UI显示
